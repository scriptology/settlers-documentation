Игровой мир разделен на три слоя карты: Слой гексов, Слой территории клана и Глобальный слой.

## Слой гексов
- Площадь: ~0,015 км²
- Количество: 33,897,029,870 гексов по всему миру
- Базовая единица для захвата территории

## Слой гексы клана
- Площадь: ~0.737 км²
- Количество: 691,776,122 гексов по всему миру
- Структура: Состоит из 49 обычных гексов
- Показывает доминирование кланов в районе

### Правила контроля
- Нужно >50% гексов для контроля клана
- Пример: 25/49 гексов = 51% = контроль клана
- Обновляется динамически при захватах
