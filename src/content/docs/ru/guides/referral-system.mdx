## Способы приглашения

В SETTLERS есть два способа пригласить игрока и сделать его своим рефералом:

### 1. Приглашение в клан
- Используйте кнопку "Пригласить" на странице клана
- Игрок присоединяется к клану и автоматически становится вашим рефералом

### 2. Приглашение друга
- Используйте ссылку "Пригласить друга" из профиля
- Игрок регистрируется как ваш реферал
